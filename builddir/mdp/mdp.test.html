mdp.test.test(filename=None, seed=None, options='', mod_loc=None)[source]¶

Run tests.

filename – only run tests in filename. If not set run all tests.
You do not need the full path, the relative path within the test directory is enough.

seed – set random seed

options – options to be passed to py.test (as a string)

mod_loc – don’t use it, it’s for internal usage
