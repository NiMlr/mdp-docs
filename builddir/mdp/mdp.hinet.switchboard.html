class mdp.hinet.switchboard.MeanInverseSwitchboard(input_dim, connections)[source]Â¶

Bases: mdp.hinet.Switchboard

Variant of Switchboard with modified inverse.

If the switchboard mapping is not injective, then the mean values are used for the inverse. Inputs that are discarded in the mapping are set to zero.

You can use this class as a mixin for other switchboard classes.
