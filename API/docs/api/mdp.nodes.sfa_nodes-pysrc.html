 98          stop_training. 
 99           
100          The delta value of a signal is a measure of its temporal 
101          variation, and is defined as the mean of the derivative squared, 
102          i.e. delta(x) = mean(dx/dt(t)^2).  delta(x) is zero if 
103          x is a constant signal, and increases if the temporal variation 
104          of the signal is bigger. 
105           
106          The eta value is a more intuitive measure of temporal variation, 
107          defined as 
108              eta(x) = t/(2*pi) * sqrt(delta(x)) 
109          If x is a signal of length 't' which consists of a sine function 
110          that accomplishes exactly N oscillations, then eta(x)=N. 
111

113          t -- Time units (e.g., t=0.01 if you sample at 100Hz) 
114          """
