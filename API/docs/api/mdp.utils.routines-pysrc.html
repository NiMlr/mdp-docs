 82      Stewart, G.W., "The efficient generation of random orthogonal 
 83      matrices with an application to condition estimators", SIAM Journal 
 84      on Numerical Analysis, 17(3), pp. 403-409, 1980. 
 85      For more information see 
 86      http://en.wikipedia.org/wiki/Orthogonal_matrix#Randomization""" 
 87      H = mdp.numx.eye(dim, dtype=dtype) 
 88      D = mdp.numx.ones((dim,), dtype=dtype)

 91          D[n-1] = mdp.numx.sign(x[0])
